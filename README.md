# Installation

Clone repository:

```
git clone https://github.com/stefank-29/ReaderApp.git
```

### Run project:

```
cd ReaderApp/
```

```

npm install

npm run dev

```

Open in browser at address:

```

http://localhost:3000/

```

### Built with:

-   React (Next.js)
